# Left Sidebar

This file explains the left sidebar shown on the blog main page after login.

---

## Purpose

The left sidebar is used only for **navigation**.
It helps users move between pages.

---

## Items Shown

- Home  
  Shows all published blogs.

- For You  
  Shows personalized blogs.

- Library  
  Shows saved and bookmarked blogs.

- Write  
  Opens the blog editor.

- Profile  
  User profile and settings.

---

## Rules

- Do not show blog suggestions
- Do not show trending content
- Do not show tags or cards
- Keep layout simple

---

## Design Notes

- Icon + text
- Small width
- Clean and minimal

---

## Behavior

- Always visible on desktop
- Hidden on mobile
- Opens from menu on mobile


# Center Blog Content

This file explains the main blog feed shown in the center of the page.

---

## Purpose

The center area is the **main content area**.
It shows all blog posts.

---

## Blog Item Structure

Each blog item shows:

- Author avatar and name
- Publish date
- Blog title
- Short description (2–3 lines)
- Tags
- Like, comment, read time, save buttons

# Right Sidebar – Suggestions

This file explains the right sidebar used for content suggestions.

---

## Purpose

The right sidebar helps users **discover content**.
It is secondary to the blog feed.

---

## Sections

---

## Recommended for You

Shows personalized blog suggestions.

### Based On

- Blogs liked by the user
- Blogs saved by the user
- Tags followed by the user

### UI

- Text-only list
- 3–5 blog titles

---

## Trending

Shows currently popular blogs.

### Based On

- Views
- Likes
- Comments
- Last 24–48 hours activity

### UI

- Numbered list
- 3–5 blog titles

---

## Popular Tags

Shows frequently used tags.

### UI

- 6–8 tags
- Click a tag to filter blog feed

---

## Rules

- Keep content lightweight
- No images
- No large cards
- Make sidebar sticky on desktop

---

## Mobile Behavior

- Hidden on mobile
- Accessible from explore or menu screen
